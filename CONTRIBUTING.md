# Contributing

Thank you for your interest in contributing to RabbitMQ Zero To Hero.

## How to Contribute

1. Fork the repository.
2. Create a feature branch.

```bash
git checkout -b feature/my-change
```

3. Make your changes.
4. Commit your work.

```bash
git commit -m "Add my contribution"
```

5. Push the branch.

```bash
git push origin feature/my-change
```

6. Open a Pull Request.

## Contribution Guidelines

* Follow the existing repository structure.
* Keep examples simple and educational.
* Maintain consistency across chapters.
* Update documentation when necessary.
* Ensure diagrams and screenshots are clearly named.

## Reporting Issues

If you find an issue, please open a GitHub issue with:

* Description of the problem
* Steps to reproduce
* Expected behavior
* Actual behavior

## Suggestions

Feature suggestions and improvements are always welcome.
