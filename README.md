## About The Repository

RabbitMQ is one of the most widely used message brokers for building scalable, reliable, and event-driven systems.

This repository is a structured learning path designed to help developers understand RabbitMQ from scratch and gradually progress toward production-grade messaging patterns used in real-world applications.

The content is organized into chapters covering both fundamental and advanced concepts, accompanied by diagrams, examples, and practical implementations using Java and Spring Boot.
